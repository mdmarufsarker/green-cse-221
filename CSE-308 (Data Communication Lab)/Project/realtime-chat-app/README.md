# Real Time Chat Application 

This is a real time chat application using

- React
- Firebase
- Emoji Picker React
- React Toastify
- Zustand
- React Router Dom
- Timeago.js

## Features

- Login Registration using username and password
- Real time chat
- Can send emojis
- Can send images
- Highlight the unread messages
- Show the last message time (not working properly)
- Logout
- Block user
